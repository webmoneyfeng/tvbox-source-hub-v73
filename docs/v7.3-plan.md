# v7.3 执行计划摘要

本项目从 v7.2 基准框架独立派生，目标是固定域名入口、多镜像快照、动态聚合兜底、免费优先。

## 验收矩阵

- `/config.json`：只显示一个点播站点 `影视点播`。
- `/agg`：推荐非空。
- `/agg?ac=videolist&t=1&pg=1`：分类列表非空。
- `/agg?ac=detail&t=1&pg=1`：FongMi 分类兼容。
- `/agg?ac=detail&ids=xxx`：详情可用。
- `/agg?wd=解说`：搜索可用。
- `/agg?f={...}`：筛选可用。
- `/status.json`、`/snapshot.json`、`/mirrors.json`：运维接口可用。
