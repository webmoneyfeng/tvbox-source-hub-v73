# v7.3 执行计划摘要

本项目从 v7.2 基准框架独立派生，当前目标设定已升级为“收费级体验 + 自运营全免费 + 全量准实时更新”：

- 入口统一为 `影视点播`
- 终局目标：顶级的实时更新 TVBox/FongMi/影视仓点播 + 直播源，可收费，满足用户一切期待，无投诉商业项目
- 自运营成本：免费优先，不买主机、不买付费 CDN、不代理大规模视频流
- 热更新只写 KV / 热索引，不触发全量 Pages 构建
- 全量快照每日一次，作为回滚与兜底
- Worker 读取静态快照时优先命中最新 GitHub raw 镜像，再回落 Pages / 自有静态域名 / 动态兜底
- 直播与点播分流 ingest，但 UI 层共用 canonical 去重
- 热点 5–15 分钟可见，全量 24 小时内可见，接受小时级时差

## 验收矩阵

- `/config.json`：只显示一个点播站点 `影视点播`
- `/agg`：推荐非空
- `/agg?ac=videolist&t=1&pg=1`：分类列表非空
- `/agg?ac=detail&t=1&pg=1`：FongMi 分类兼容
- `/agg?ac=detail&ids=xxx`：详情可用
- `/agg?wd=解说`：搜索可用
- `/agg?f={...}`：筛选可用
- `/status.json`、`/snapshot.json`、`/mirrors.json`：运维接口可用
